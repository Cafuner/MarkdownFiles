## ElasticSearch中的查询语句分类：

1. 全文检索
    1. match_all查询
    2. match查询
    3. multi-match查询
2. 精确查询
    1. term查询
    2. range查询
3. 地理查询
    1. geo_bounding_box查询
    2. geo_distance查询
4. 复合查询
    1. Function Score Query
    2. Boolean Query
        1. must
        2. should
        3. must_not
        4. filter
